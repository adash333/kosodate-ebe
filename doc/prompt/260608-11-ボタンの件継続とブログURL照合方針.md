## プロンプト

> だめでした。また後で、check againしたいと思います。ボタンの件を続けてください。ブログURLは、https://risan.jpn.org/ の中で、動画投稿日と、ブログ投稿日が一致しています。動画投稿日は、動画リストから確認できます。

## 概要

HTTPS は後回しにして、論文解説ボタン（ブログリンク）の作業を継続するよう指示。ブログURLの特定方法として「動画投稿日＝ブログ投稿日」「動画投稿日は動画リストで確認可能」というヒントを提示。最終的には WordPress REST API のタイトル内『著者, 年』で照合した。
