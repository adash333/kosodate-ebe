# 合併症5記事に近視分類表追加とmainマージ

## 概要

前セッションで作成した `claude/myopia-author-credentials-wv8m9q` ブランチの未コミット変更
（`scripts/daily-deploy.sh` 修正）をコミット・プッシュし、`main` にマージした。
また、強度近視の合併症を扱う5記事の冒頭に近視度数分類表（正視〜超強度近視）を追加した。

## 実施内容

### 1. daily-deploy.sh のコミット＆プッシュ

前セッションで編集済みだった `scripts/daily-deploy.sh` をコミットし、
`claude/myopia-author-credentials-wv8m9q` ブランチにプッシュ。

- 変更内容: `git pull --ff-only` → `git fetch origin main && git checkout main && git reset --hard origin/main`
- 理由: VM にローカル変更が残っていると `git pull --ff-only` が失敗するため、`reset --hard` で確実に同期するよう修正

### 2. claude ブランチを main にマージ

`lib/articles.ts` でコンフリクトが発生（main 側の seed-edof 記事 2026-07-08 と、
claude 側の8記事 2026-06-30〜2026-07-07 が同じ位置に挿入されていた）。
日付順に並べ直してコンフリクトを解消し、マージコミットを main に push。

マージに含まれた変更:
- `app/guide/` 配下8記事の新規追加（オルソケラトロジー適応・禁忌、近視性黄斑症、網膜剥離、緑内障、白内障、近視性牽引黄斑症・強膜ぶどう腫、視力と職業、運転免許と視力基準）
- `lib/references.ts`: 12件の参考文献追加
- `app/about/page.tsx`: 運営者表記を「子どもの近視ナビ（運営: 合同会社５マイクロ）」に修正
- `app/layout.tsx`: WebSite JSON-LD に publisher フィールド追加（Google検索サイト名の統一）
- `scripts/daily-deploy.sh`: git pull 修正

### 3. 合併症5記事に近視分類表を追加

共通コンポーネント `components/MyopiaGradeTable.tsx` を新規作成し、
以下5記事の冒頭（最初の本文段落の直前）に `<MyopiaGradeTable />` を挿入。

- `app/guide/myopic-maculopathy/page.tsx` — 近視性黄斑症とは
- `app/guide/retinal-detachment-myopia/page.tsx` — 網膜剥離と近視のリスク
- `app/guide/myopia-glaucoma/page.tsx` — 緑内障と近視の関係
- `app/guide/myopia-cataract/page.tsx` — 白内障と近視
- `app/guide/myopic-traction-maculopathy/page.tsx` — 近視性牽引黄斑症・強膜ぶどう腫とは

#### 分類表の内容（IMI 2019 基準）

| 分類 | SE の目安 |
|---|---|
| 正視 | −0.50D 未満（プラス側含む） |
| 軽度近視 | −0.50D 〜 −3.00D 未満 |
| 中等度近視 | −3.00D 〜 −6.00D 未満 |
| 強度近視 | −6.00D 以下 |
| 超強度近視（病的近視） | −8.00D 以下（または眼軸長≥26.5mm+眼底変化） |
