# Search Consoleサイトマップ送信手順とAdSense審査状況の確認

## 概要

前作業（260624-01のAdSense対策）に続き、Google Search Console でのサイトマップ送信手順を案内した。
あわせてユーザー提供のスクリーンショットから、AdSense が今日の改修後の状態で
すでに「審査待ち」になっていることを確認した。コード変更は行っていない（運用・案内のみ）。

## 実施内容

### 1. サイトマップの所在確認

- サイトマップ: `https://5micro.net/sitemap.xml`（86 URL）。
- `public/robots.txt` からも `Sitemap: https://5micro.net/sitemap.xml` で参照済み。
- 独自ドメイン `5micro.net`（`public/CNAME`）。

### 2. Search Console サイトマップ送信手順の案内

1. ブラウザで `https://5micro.net/sitemap.xml` がXML表示されることを確認（デプロイ反映に数分）。
2. https://search.google.com/search-console でプロパティ `5micro.net` を選択。
3. 左メニュー「サイトマップ」→ 入力欄に `sitemap.xml` → 送信。
4. ステータス「成功しました」「検出されたURL: 86」を確認。
5. 「取得できませんでした」でも数時間〜1日で成功することが多い旨を補足。
- プロパティ未登録時は「URLプレフィックス＋HTMLタグ」方式を推奨。
  確認用 meta タグをもらえれば `index.html` に埋め込み main へpush対応可能、と案内。

### 3. AdSense審査状況の確認（スクリーンショットより）

- 画面は AdSense（Search Consoleとは別サービス）。
- `5micro.net` は「準備中」で、
  - 「サイトの所有権を確認してください」→ 完了（✅）
  - 「審査待ち」→ 有効（✅）／リクエスト日時: 2026年6月24日 17:30
- → 今日の改修（プリレンダリング・トップの読み物・著者プロフィール・出典リンク）を
  反映した状態で AdSense 審査がすでに進行中。追加申請操作は不要。

## 今後の運用メモ

- 審査結果を待つ（通常数日、最大2〜4週間）。
- 待機中に Search Console でサイトマップ送信＋主要URL（トップ／`/articles`／記事数本）の
  インデックス登録リクエストを行うとプラス材料になる。
- サイトマップ送信でエラーが出た場合は画面を共有して対応する。
