# 睡眠記事の追加と AdSense 再対策（プライバシーポリシー刷新・著者ページ新設・関連記事）

## 概要

Obsidian の論文リストをもとに新規公開動画（#167-171）を動画一覧・3ステップ相談室へ反映したのち、
新しい睡眠テーマの読み物を1本追加した。さらに Google AdSense に再び「有用性の低いコンテンツ」で
却下されたスクリーンショットを受け、YMYL/E-E-A-T と規約準拠を高める一連の改修を `main` に直 push した。
プライバシーポリシーの AdSense 準拠への全面刷新、著者詳細ページ `/author` の新設、
各記事末尾への関連記事リンク追加が主な内容。あわせて著者経歴の表記も事実に即して修正した。

## 実施内容

### 1. 新規動画（#167-171）の反映（前半セッション）

- `data/enrichment.json`: #167〜171 に YouTube URL・ブログ URL を追記。
- `data/advice.json`: `node scripts/merge-data.mjs` を直接実行して再生成（161→169本）。
  - 注意: `npm run data` は vault パス（`../../東大理三夫婦の子育てエビデンス研究所`）を
    解決できず `_raw_extract.json` を 0 件で上書き破壊するため使用しない。merge-data.mjs のみ実行。
- `data/taxonomy.json`: 3ステップ相談室で新動画が上位に出るようタグを追加。
  - `dev-asd` に「応答」「社会スキル」、`dev-adhd` に「有酸素運動」を追加。

### 2. 睡眠の読み物を新規追加

- `src/articles.ts`: 記事 `nekashire`（寝かしつけのルーティンとスクリーン）を追加。
  - publish: `2026-07-12`（既存最終記事 koukishin 7/11 の翌日）。
  - 構成: ①就寝ルーティン（Mindell 2009/2015）②ブルーライトとメラトニン（Chang 2014）
    ③ADHD・ASD 児の睡眠と親教育（Malow 2014）。各出典に原典リンクを付与。

### 3. プライバシーポリシー・免責事項を AdSense 準拠に全面刷新（最重要）

- `src/components/Legal.tsx`: `Disclaimer` と `PrivacyPolicy` に分割し内容を大幅拡充。
  - プライバシーポリシー: AdSense Cookie・DoubleClick Cookie の明示的開示、
    パーソナライズ広告のオプトアウト3手段（Google 広告設定 / aboutads.info / ブラウザ設定）、
    Google Analytics(GA4) 開示とオプトアウトアドオン、第三者提供・保管期間・
    未成年者保護・改定条項・運営者連絡先（会社住所）を追加。
  - 免責事項: 情報の性質／専門家相談／外部リンク／著作権／サービス変更の5セクション構成に拡充。
