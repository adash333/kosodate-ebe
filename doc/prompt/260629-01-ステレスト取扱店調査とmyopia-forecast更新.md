# ステレスト取扱店調査と myopia-forecast 更新

## 概要

stellest.jp を起点にステレスト（Essilor Stellest）の日本取扱店を調査し、
myopia-forecast の眼科クリニック一覧・眼鏡店一覧を大幅に更新した。
ステレストは 2026年6月11日に日本上陸した小児近視進行抑制レンズ。

## 実施内容

### 1. ステレスト製品調査

- stellest.jp は環境プロキシポリシーで直接アクセス不可
- Web 検索でステレストの概要・日本取扱店情報を収集
  - H.A.L.T. テクノロジー（1,021個のマイクロレンズレット）
  - 2年間で平均 71% の近視進行抑制（臨床試験）
  - 2026年6月11日 日本発売、ニコン・エシロールが展開
